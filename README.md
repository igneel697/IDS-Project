# Real-Time Network Intrusion Detection System

Academic project implementing hybrid IDS using rule-based and ML approaches.

## Tech Stack
- Backend: Python, Flask
- Frontend: React
- Database: MySQL
- ML: scikit-learn
- Packet Capture: Wireshark/tshark
